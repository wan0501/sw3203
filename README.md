# SW3203 单向升降压电源板

## 项目简介
这是一个基于 **SW3203** 控制器的单向升降压电源硬件项目，支持宽范围直流输入与可调输出，适合用于便携式供电、电池升降压、电源模块验证以及高功率 DC-DC 方案开发。

项目当前主要用于保存原理图、设计资料和后续调试记录，方便版本管理、打样和测试。

## 主要参数
- 输入电压：`3V ~ 28V`
- 输出电压：`3V ~ 22V`
- 最大输入电流：`12A`
- 最大输出电流：`6.85A`

## 方案特点
- 基于 **SW3203** 的单向升降压控制方案
- 采用四开关 Buck-Boost 主功率拓扑
- 支持输入电压低于、等于或高于输出电压时稳定工作
- 支持较高输入/输出电流
- 板上带有 **USB Type-C** 相关输入电路
- 适合做高功率可调 DC-DC 电源方案验证

## 电路结构说明
整个设计大致分为以下几个部分：

### 1. 输入接口与保护
输入端支持宽范围直流输入，原理图中包含输入接口、二极管保护以及输入滤波电容，用于抑制纹波并提高系统稳定性。

### 2. USB-C 触发/诱骗电路
电路中包含 USB Type-C 接口及相关触发芯片，用于从支持 Type-C/PD 的电源适配器获取所需输入电压，方便作为前级供电输入。

### 3. 主功率级
主功率回路采用四开关 Buck-Boost 结构，通过电感与 MOSFET 协同工作，实现：
- 降压模式
- 升压模式
- 升降压自动过渡

该结构适用于输入输出范围交叉较大的应用场景。

### 4. 控制与反馈网络
控制器通过输出电压反馈、补偿网络及采样信号调节功率级工作状态，实现稳压输出。原理图中预留了反馈分压和环路补偿元件，便于后续调试优化。

## 应用方向
- 可调直流电源
- 电池升降压供电系统
- USB-C 输入电源转换模块
- 便携式高功率供电设备
- 嵌入式设备电源前端
