# 单向升降压电源项目

## 项目简介
这是一个基于四开关 Buck-Boost 拓扑的单向升降压电源设计项目，支持较宽输入输出范围，适合移动电源、电池供电设备、可调电源模块等应用场景。

本项目目前主要包含原理图 PDF，用于电路设计说明、器件连接参考和后续 PCB/Layout 开发。

## 主要参数
- 输入电压：`3V ~ 28V`
- 输出电压：`3V ~ 22V`
- 最大输入电流：`12A`
- 最大输出电流：`6.85A`

## 功能特点
- 支持输入电压低于、等于或高于输出电压时稳定工作
- 采用单向升降压结构
- 具备较大电流输出能力
- 带有 USB Type-C 输入相关电路
- 适合做高功率可调 DC-DC 电源方案验证

## 电路说明
本设计整体由以下几个部分组成：

### 1. 输入接口
输入端支持较宽范围直流电压输入，原理图中包含输入端保护和滤波电路，用于降低输入纹波并提高系统稳定性。

### 2. USB-C 触发/协商部分
电路中包含 USB Type-C 相关接口和诱骗/触发芯片，用于获取所需输入电压，方便接入支持 PD/Type-C 的电源适配器。

### 3. 主功率级
主功率回路采用四开关 Buck-Boost 拓扑，通过电感和 MOSFET 组合实现升压、降压及升降压切换，满足宽范围输入输出需求。

### 4. 控制与反馈
控制芯片通过采样输出电压和相关反馈信号调节占空比，实现输出稳压。原理图中预留了反馈分压及补偿网络，用于稳定环路。

## 应用方向
- 可调直流电源
- 电池升降压供电系统
- USB-C 供电转换模块
- 便携式高功率供电设备
- 嵌入式设备电源前端
